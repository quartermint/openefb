    **WeatherService.swift** — `actor WeatherService: WeatherServiceProtocol`:

    Properties:
    - `private var cache: [String: WeatherCache] = [:]` // in-memory cache keyed by station ID
    - `private let session: URLSession`
    - `private static let metarURL = "https://aviationweather.gov/api/data/metar"`
    - `private static let tafURL = "https://aviationweather.gov/api/data/taf"`
    - `private static let cacheExpiry: TimeInterval = 900` // 15 minutes

    Methods:
    - `init()` — Create URLSession with default config

    - `func fetchMETAR(for stationID: String) async throws -> WeatherCache`:
      1. Check cache — if entry exists and `age < cacheExpiry`, return cached
      2. Build URL: `"\(Self.metarURL)?ids=\(stationID)&format=json"`
      3. Fetch with URLSession, decode JSON array
      4. Parse NOAA response JSON. The NOAA API returns an array of objects with fields: `icaoId`, `rawOb`, `temp`, `dewp`, `wdir`, `wspd`, `wgst`, `visib`, `clouds` (array of `{cover, base}`), `obsTime`, `fltCat`
      5. Build WeatherCache from parsed data:
         - `stationID` = `icaoId`
         - `rawMETAR` = `rawOb`
         - `temperature` = `temp`
         - `dewpoint` = `dewp`
         - `windDirection` = `wdir`
         - `windSpeed` = `wspd`
         - `windGust` = `wgst` (nullable)
         - `visibility` = `visib`
         - `ceiling` = lowest cloud base with cover "BKN" or "OVC"
         - `flightCategory` = determine from ceiling and visibility per FAA: VFR (ceiling >3000 AND vis >5), MVFR (ceiling 1000-3000 OR vis 3-5), IFR (ceiling 500-999 OR vis 1-2), LIFR (ceiling <500 OR vis <1). Or use the `fltCat` field from NOAA if available.
         - `observationTime` = parse `obsTime` with `ISO8601DateFormatter`
         - `fetchedAt` = `Date()`
      6. Store in cache, return

    - `func fetchTAF(for stationID: String) async throws -> String`:
      1. Build URL: `"\(Self.tafURL)?ids=\(stationID)&format=json"`
      2. Fetch, decode, return `rawTAF` field from first array element
      3. Throw `EFBError.weatherFetchFailed` on error

    - `func fetchWeatherForStations(_ stationIDs: [String]) async throws -> [WeatherCache]`:
      1. Filter out recently cached stations (< 15 min old)
      2. Batch remaining into groups of 40 (to stay well under URL length limits)
      3. For each batch: build URL `"\(Self.metarURL)?ids=\(batch.joined(separator:","))&format=json"`
      4. Fetch and parse all, update cache
      5. Return all weather data (cached + freshly fetched)

    - `nonisolated func cachedWeather(for stationID: String) -> WeatherCache?`:
      Note: For the actor pattern, this needs to be `func cachedWeather(for stationID: String) async -> WeatherCache?` instead, or make cache access through a method. Adjust protocol if needed.

    **WeatherBadge.swift** — `struct WeatherBadge: View`:

    Per UI-SPEC: Capsule shape, 8pt horizontal / 4pt vertical padding, color-coded dot + text.
    Per user decision: yellow when >30 minutes old, red when >60 minutes old.
    Per UI-SPEC copywriting: "<1 min" / "15 min" / "1h 30m" format, "STALE" when >2h.

    ```swift
    struct WeatherBadge: View {
        let observationTime: Date?

        private var minutesAgo: Int {
            guard let obs = observationTime else { return 0 }
            return Int(Date().timeIntervalSince(obs) / 60)
        }

        private var displayText: String {
            guard observationTime != nil else { return "N/A" }
            if minutesAgo >= 120 { return "STALE" }
            if minutesAgo < 1 { return "<1 min" }
            if minutesAgo < 60 { return "\(minutesAgo) min" }
            let hours = minutesAgo / 60
            let mins = minutesAgo % 60
            if mins == 0 { return "\(hours)h" }
            return "\(hours)h \(mins)m"
        }

        private var badgeColor: Color {
            if minutesAgo >= 120 { return .gray }    // STALE per UI-SPEC
            if minutesAgo > 60 { return .red }       // >60 min per user decision
            if minutesAgo > 30 { return .yellow }    // >30 min per user decision
            return .secondary                        // fresh
        }

        var body: some View {
            Text(displayText)
                .font(.caption2)
                .fontWeight(.medium)
                .padding(.horizontal, 8)
                .padding(.vertical, 4)
                .background(badgeColor.opacity(0.2))
                .foregroundColor(badgeColor)
                .clipShape(Capsule())
        }
    }
    ```

    struct FlightCategoryDot: View {
