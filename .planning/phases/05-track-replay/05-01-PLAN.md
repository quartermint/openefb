    let flightID: UUID
    let timestamp: Date
    let text: String
    let confidence: Double

    Create ReplayEngine.swift -- @Observable @MainActor final class ReplayEngine. This is the single source of truth time model for synchronized track replay (per REPLAY-01, REPLAY-02).

    Properties:
    - `currentPosition: TimeInterval = 0` -- seconds from recording start (THE source of truth)
    - `isPlaying: Bool = false`
    - `playbackSpeed: Float = 1.0` -- 1x, 2x, 4x, 8x per user decision
    - `audioMuted: Bool = false` -- true at 4x/8x because AVAudioPlayer max rate is 2.0
    - `currentCoordinate: CLLocationCoordinate2D` -- interpolated position
    - `currentAltitude: Double = 0` -- feet MSL at current position
    - `currentSpeed: Double = 0` -- knots at current position
    - `currentHeading: Double = 0` -- degrees true at current position
    - `currentTranscriptIndex: Int? = nil` -- index into transcriptSegments array
    - `totalDuration: TimeInterval` -- computed from first/last track point timestamps
    - `phaseMarkerFractions: [(phase: String, fraction: Double)]` -- for scrub bar markers

    Private data:
    - `trackPoints: [TrackPointRecord]`
    - `transcriptSegments: [TranscriptSegmentRecord]`
    - `phaseMarkers: [PhaseMarkerRecord]`
    - `audioPlayer: AVAudioPlayer?`
    - `playbackTimer: Timer?`
    - `recordingStartTime: Date`

    Methods:
    - `loadFlight(flightID: UUID, recordingDB: RecordingDatabase, audioURL: URL?) throws`
      - Fetches trackPoints, transcriptSegments, phaseMarkers from recordingDB
      - Sets recordingStartTime = trackPoints.first?.timestamp ?? Date()
      - Computes phaseMarkerFractions
      - If audioURL exists and file is present, creates AVAudioPlayer with enableRate = true, calls prepareToPlay()
      - Calls updateDerivedState()

    - `play()`
      - Sets isPlaying = true
      - Syncs audio to currentPosition, starts audio if not muted
      - Starts 20Hz Timer (0.05s interval) calling tick()

    - `pause()`
      - Sets isPlaying = false
      - Pauses audioPlayer
      - Invalidates playbackTimer

    - `seekTo(_ position: TimeInterval)`
      - Clamps position to 0...totalDuration
      - Sets currentPosition
      - Syncs audioPlayer.currentTime = currentPosition
      - Calls updateDerivedState()

    - `setSpeed(_ speed: Float)`
      - Sets playbackSpeed = speed
      - If speed <= 2.0: audioPlayer.rate = speed, audioMuted = false, audioPlayer.volume = 1.0
      - If speed > 2.0: audioMuted = true, audioPlayer.volume = 0 (keep audio advancing at 2x so it stays roughly in sync)
      - Per research: AVAudioPlayer rate caps at 2.0

    - `func interpolatedPosition(at time: TimeInterval) -> (coordinate: CLLocationCoordinate2D, heading: Double, altitude: Double, speed: Double)`
      - Compute targetTime = recordingStartTime + time
      - Binary search / firstIndex for bracketing track points
      - Linear interpolation of lat, lon, heading, altitude, speed between bracketing points
      - Edge cases: before first point returns first, after last returns last

    - Private `tick()`
      - Advance currentPosition by 0.05 * Double(playbackSpeed)
      - If currentPosition >= totalDuration: clamp, pause()
      - Call updateDerivedState()
      - Every 100 ticks (~5 seconds), if audio is playing and not muted: drift correction -- read audioPlayer.currentTime, if abs(currentPosition - audioPlayer.currentTime) > 0.2 seconds, seek audio to currentPosition

    - Private `updateDerivedState()`
      - Call interpolatedPosition(at: currentPosition) to get coordinate, heading, altitude, speed
      - Set currentCoordinate, currentHeading, currentAltitude, currentSpeed
      - Find currentTranscriptIndex: first segment where audioStartTime <= currentPosition < audioEndTime

    - Private `syncAudioToPosition()`
      - audioPlayer?.currentTime = currentPosition

    - `cleanup()` -- invalidate timer, stop audio, nil out audioPlayer

    For tests, create ReplayEngineTests.swift using Swift Testing (`import Testing`). Create a helper function `makeTestRecordingDB()` that:
    - Creates a temp file for DatabasePool (pattern from Phase 4: temp file, not :memory:)
    - Creates a RecordingDatabase(dbPool:) instance
    - Inserts 10 TrackPointRecords for a test flightID, timestamps 1 second apart, with varying lat/lon/alt/speed
    - Inserts 3 TranscriptSegmentRecords with audioStartTime/audioEndTime spanning different time ranges
    - Inserts 3 PhaseMarkerRecords (taxi, takeoff, cruise) at different startTimestamps
    - Returns (recordingDB, flightID)

    Tests do NOT need AVAudioPlayer (pass audioURL: nil to loadFlight). The engine works without audio.
