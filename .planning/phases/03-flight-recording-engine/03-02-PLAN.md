    **1. Create AudioRecorder (efb-212/Services/Recording/AudioRecorder.swift):**

    Actor conforming to `AudioRecorderProtocol`. Key properties:

    ```swift
    actor AudioRecorder: AudioRecorderProtocol {
        private var audioEngine: AVAudioEngine?
        private var outputFile: AVAudioFile?
        private var _isRecording = false
        private var _audioLevel: Float = -160
        private var flightID: UUID?

        // Callback for streaming PCM buffers to transcription service
        var onBufferAvailable: ((AVAudioPCMBuffer, AVAudioTime) -> Void)?

        // Callback for gap markers (interruptions)
        var onInterruptionGap: ((InterruptionGapReason) -> Void)?

        // Notification observers
        private var interruptionObserver: NSObjectProtocol?
        private var routeChangeObserver: NSObjectProtocol?
        private var didBecomeActiveObserver: NSObjectProtocol?
        private var isPausedByInterruption = false
    }
    ```

    **Audio session configuration** (call before engine start):
    ```swift
    private func configureAudioSession(profile: AudioQualityProfile) throws {
        let session = AVAudioSession.sharedInstance()
        try session.setCategory(
            .playAndRecord,
            mode: .voiceChat,
            options: [.allowBluetooth, .allowBluetoothA2DP, .defaultToSpeaker, .mixWithOthers]
        )
        try session.setPreferredSampleRate(profile.sampleRate)
        try session.setPreferredIOBufferDuration(0.05)  // 50ms buffer
        try session.setPreferredInputNumberOfChannels(1)  // mono
        try session.setActive(true)
    }
    ```

    **startRecording implementation:**
    1. Create recordings directory at `Application Support/efb-212/recordings/`
    2. Create output URL: `{recordingsDir}/{flightID}.m4a`
    3. Call `configureAudioSession(profile:)`
    4. Create `AVAudioEngine()`, get `inputNode`
    5. Get `recordingFormat = inputNode.outputFormat(forBus: 0)`
    6. Guard `recordingFormat.channelCount > 0` else throw `EFBError.audioSessionFailed` (SFR pattern: prevents crash on simulator)
    7. Create `AVAudioFile(forWriting: outputURL, settings: [AVFormatIDKey: kAudioFormatMPEG4AAC, AVSampleRateKey: profile.sampleRate, AVNumberOfChannelsKey: 1, AVEncoderBitRateKey: profile.bitRate])`
    8. Install tap on inputNode (bufferSize: 4096, format: recordingFormat):
       - Write buffer to outputFile: `try? outputFile.write(from: buffer)`
       - Call `onBufferAvailable?(buffer, time)` for transcription streaming
       - Compute RMS for audio level meter: `_audioLevel = 20 * log10(rms)` where rms = sqrt(mean of squares of samples in buffer)
    9. `audioEngine.prepare(); try audioEngine.start()`
    10. Set `_isRecording = true`
    11. Register interruption/route change observers

    **stopRecording implementation:**
    1. Remove tap from inputNode
    2. Stop audio engine
    3. Set `_isRecording = false`
    4. Remove notification observers
    5. Deactivate audio session: `try? AVAudioSession.sharedInstance().setActive(false, options: .notifyOthersOnDeactivation)`
    6. Return output file URL

    **Interruption handling** (per SFR CockpitAudioEngine pattern):
    - Register `NotificationCenter.default.addObserver` for `AVAudioSession.interruptionNotification`
    - On `.began`: call `pauseRecording()`, set `isPausedByInterruption = true`, call `onInterruptionGap?(.phoneCall)` (or determine reason from audio route)
    - On `.ended` with `.shouldResume`: call `resumeRecording()`
    - Register `UIApplication.didBecomeActiveNotification` as fallback: if `isPausedByInterruption` is true, attempt `resumeRecording()`
    - Register `AVAudioSession.routeChangeNotification`: on `.oldDeviceUnavailable` (headphone disconnect), call `onInterruptionGap?(.headphoneDisconnect)` -- audio continues via built-in mic per `.defaultToSpeaker`
