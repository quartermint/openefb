    **1. Create AviationVocabularyProcessor (efb-212/Services/Recording/AviationVocabulary.swift):**

    A struct with a `func process(_ text: String) -> String` method that applies regex-based corrections in order. Patterns derived from SFR's AviationVocabularyProcessor (448 lines of proven patterns):

    ```swift
    struct AviationVocabularyProcessor: Sendable {
        func process(_ text: String) -> String {
            var result = text
            result = processNNumbers(result)
            result = processFrequencies(result)
            result = processAltitudes(result)
            result = processHeadings(result)
            result = processRunways(result)
            result = processSquawkCodes(result)
            result = processATIS(result)
            result = processPhoneticDigits(result)
            return result
        }
    }
    ```

    Key regex patterns:
    - **N-numbers:** `november\s+(\w+)` -> `N\1` (with digit word-to-number conversion: "seven" -> "7", "niner" -> "9", etc.)
    - **Phonetic digits:** Map: zero=0, one=1, two/to=2, tree/three=3, four/fower=4, fife/five=5, six=6, seven=7, eight=8, niner/nine=9 -- only when in aviation context (after "squawk", "heading", "runway", "flight level", "altitude", frequency patterns)
    - **Frequencies:** `one\s+(one|two|three)\s+(zero|one|two|...)\s+point\s+(...)` -> `1X0.XX` format
    - **Altitudes:** `(altitude|maintain|descend|climb)\s+(word numbers)` -> numeric with comma formatting
    - **Flight levels:** `flight level\s+(three|two|one)\s+(zero|one|...)\s+(zero|one|...)` -> `FL350` format
    - **Headings:** `heading\s+(word numbers)` -> `heading 360` format
    - **Runways:** `runway\s+(word numbers)\s*(left|right|center)?` -> `Runway 27L` format
    - **Squawk codes:** `squawk\s+(word numbers){4}` -> `Squawk 1200` format
    - **ATIS:** `(information|atis)\s+(alfa|bravo|charlie|delta|echo|foxtrot|golf|hotel|india|juliet|kilo|lima|mike|november|oscar|papa|quebec|romeo|sierra|tango|uniform|victor|whiskey|xray|yankee|zulu)` -> `ATIS Information B` (first letter)

    Helper: `wordToDigit(_ word: String) -> String?` maps spoken numbers to digits.

    **2. Create TranscriptionService (efb-212/Services/Recording/TranscriptionService.swift):**

    Actor conforming to `TranscriptionServiceProtocol`. Uses SpeechAnalyzer (iOS 26) as primary with SFSpeechRecognizer fallback.

    ```swift
    actor TranscriptionService: TranscriptionServiceProtocol {
        private let recordingDB: RecordingDatabase
        private let vocabularyProcessor = AviationVocabularyProcessor()
        private var transcriptionTask: Task<Void, Never>?
        private var _isTranscribing = false
        private var flightID: UUID?
        private var currentPhaseProvider: (() async -> FlightPhaseType)?

        // Callback for UI updates (volatile + final)
        var onTranscriptUpdate: ((TranscriptDisplayItem) -> Void)?

        // Buffer input for audio streaming
        private var bufferContinuation: AsyncStream<AudioBufferInput>.Continuation?
        private var bufferStream: AsyncStream<AudioBufferInput>?

        enum TranscriptionEngine {
            case speechAnalyzer  // iOS 26 SpeechAnalyzer
            case sfSpeechRecognizer  // Fallback
        }
    }
    ```

    **startTranscription flow:**
    1. Create `AsyncStream<AudioBufferInput>` pair for buffer input
    2. Check `SpeechTranscriber.installedLocales.contains(Locale(identifier: "en-US"))`:
       - If yes: Create `SpeechTranscriber(locale: Locale("en-US"), reportingOptions: [.volatileResults], attributeOptions: [.audioTimeRange])`, create `SpeechAnalyzer(modules: [transcriber])`, get `bestAvailableAudioFormat`
       - If no: Create `SFSpeechRecognizer(locale: Locale("en-US"))` with `requiresOnDeviceRecognition = true`, `shouldReportPartialResults = true`, `addsPunctuation = true`
    3. Start transcription task that consumes results:
       - For each result: get text, run through `vocabularyProcessor.process()`
       - If `result.isFinal`:
         - Create `TranscriptSegmentRecord` with processed text, confidence, audio time range, current flight phase
         - Write to GRDB: `recordingDB.insertTranscript(segment)`
         - Call `onTranscriptUpdate?(TranscriptDisplayItem(text: processed, isVolatile: false))`
       - If volatile (not final):
         - Call `onTranscriptUpdate?(TranscriptDisplayItem(text: processed, isVolatile: true))`

    **feedBuffer method** (called from AudioRecorder's onBufferAvailable):
    ```swift
    func feedBuffer(_ buffer: AVAudioPCMBuffer, time: AVAudioTime) {
        bufferContinuation?.yield(AudioBufferInput(buffer: buffer, time: time))
    }
    ```

    For SpeechAnalyzer path: convert buffer to SpeechAnalyzer's preferred format using `AVAudioConverter` if needed (per research pitfall 1), then yield as `AnalyzerInput`.

    For SFSpeechRecognizer path: append buffer to `SFSpeechAudioBufferRecognitionRequest`.
